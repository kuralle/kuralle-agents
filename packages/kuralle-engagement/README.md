# @kuralle-agents/engagement

Channel-agnostic engagement layer for Kuralle agents: window-safe outbound, smart-send, interactive fidelity, handoff, consent, and proactive messaging (see RFC REQ-22 in [`rfcs/whatsapp-engagement/`](../../rfcs/whatsapp-engagement/)).

**Status:** scaffold only — the public API lands incrementally across the sprint roadmap (`sprints/sprint-0/PLAN.md`).
