💊 Pharmacy Rx Agent

A WhatsApp-style pharmacy ordering agent on Cloudflare Durable Objects. It reads a prescription image, checks inventory, manages a persistent cart, and confirms the order after a payment link is hit — demonstrating multimodal intake and durable human-in-the-loop checkout.

Chat endpoint (CF Agents protocol)

One Durable Object per thread (tenant). Connect a CF Agents client to:

/agents/pharmacy-agent/<thread-id>

Send a user message whose parts include a text part and a file part (the prescription image URL + mediaType).

Payment callback

GET /pay/<token>

Hitting the link delivers the durable payment signal and resumes the order.
